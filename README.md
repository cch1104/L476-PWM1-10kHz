# L476-PWM1-10kHz
📌 STM32 PWM Signal Generation (TIM2)
📖 Overview

This project demonstrates how to generate a PWM (Pulse Width Modulation) signal using TIM2 Channel 2 on an STM32 microcontroller with HAL drivers.

The PWM signal is configured with:

Frequency: 10 kHz
Duty Cycle: 50%

This is a fundamental embedded systems project commonly used in:

Motor control
LED dimming
Signal generation
DAC simulation
⚙️ Hardware Requirements
STM32 board (e.g., NUCLEO-L476RG)
Oscilloscope / Logic Analyzer (recommended)
Jumper wires
🧠 How It Works

PWM is generated by configuring a timer in output compare mode.

Key Parameters:
Prescaler = 79
Period = 99
Pulse = 50
